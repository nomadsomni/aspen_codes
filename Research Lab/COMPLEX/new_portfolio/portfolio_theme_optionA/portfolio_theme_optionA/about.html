This space is a running log of experiments in interface design, visual systems, and interactive worlds. Each project is treated as a self‑contained environment: noir‑tinted, high contrast, tuned for both desktop and mobile.

Under the hood: lightweight HTML, Bootstrap skeleton, custom glass/ember styling, and WebGL scenes dropped in as live modules.

Design Stack

  • Minimal color palette — orange, charcoal, white.
  • Glass modules with soft blur & heavy shadows.
  • Typography tuned for high readability on dark.

Technical Stack

  • Vanilla HTML + Bootstrap grid.
  • Custom CSS layer for the orange/glass aesthetic.
  • WebGL modules embedded via iframe for live scenes.
